Jurusan di Sekolah Tinggi Teknologi Cipasung

Teknik Informatika

Jurusan Teknik Informatika di Sekolah Tinggi Teknologi Cipasung menawarkan pendidikan yang berfokus pada pengembangan keterampilan dan pengetahuan dalam bidang komputasi, pemrograman, sistem informasi, serta teknologi jaringan. Kurikulum dirancang untuk mempersiapkan mahasiswa menghadapi tantangan di dunia teknologi informasi yang terus berkembang.

Mata Kuliah Utama

  • Algoritma dan Struktur Data
  • Sistem Operasi
  • Pemrograman Web dan Mobile
  • Jaringan Komputer
  • Database dan Sistem Informasi
  • Kecerdasan Buatan (AI)

Prospek Karir

  • Software Developer
  • Database Administrator
  • System Analyst
  • Network Engineer
  • Cybersecurity Specialist
  • Data Scientist
  • Game Developer

Laboratorium dan Fasilitas

  • Laboratorium Komputer dengan perangkat lunak pemrograman terkini.
  • Akses ke ruang server untuk mempelajari jaringan dan keamanan komputer.

Kegiatan Mahasiswa

  • Himpunan Mahasiswa Teknik Informatika (HMTI): Menyelenggarakan seminar teknologi, hackathon, dan workshop pemrograman.
  • Kelompok Studi Pemrograman (KSP): Mendalami pengembangan aplikasi open-source dan riset teknologi.

jumlah mahasiswa Informatika

  • Jurusan Teknik Informatika: 500 mahasiswa

Teknik Industri

Jurusan Teknik Industri di STT Cipasung bertujuan untuk mempersiapkan mahasiswa dalam bidang pengelolaan sistem industri, optimalisasi proses produksi, serta perencanaan strategis dalam dunia manufaktur dan jasa. Mahasiswa akan mempelajari bagaimana meningkatkan efisiensi, produktivitas, dan kualitas produk dalam suatu sistem industri.

Mata Kuliah Utama

  • Perancangan Sistem Kerja
  • Manajemen Operasi dan Produksi
  • Manajemen Rantai Pasokan (Supply Chain)
  • Pengendalian Kualitas (Quality Control)
  • Teknologi Manufaktur
  • Statistika Industri

Prospek Karir

  • Production Manager
  • Quality Control Engineer
  • Supply Chain Manager
  • Industrial Engineer
  • Project Manager
  • Operations Analyst

Laboratorium dan Fasilitas

  • Laboratorium Ergonomi dan Perancangan Kerja untuk mempelajari efisiensi kerja dan ergonomi.
  • Laboratorium Manufaktur dan Perancangan Sistem Produksi menggunakan perangkat simulasi industri.

Kegiatan Mahasiswa

  • Himpunan Mahasiswa Teknik Industri (HMTI): Menyelenggarakan pelatihan, seminar, dan kunjungan industri.
  • Industrial Engineering Club (IEC): Komunitas yang mendalami riset dan proyek inovasi di bidang industri.

Jumlah Mahasiswa Industri

  • Jurusan Teknik Industri: 300 mahasiswa
